This week, we meet a few design terms that help us both:

1. appraise design qualities of a given web property
2. understand which to employ and how to use them in our sites

## Learning Objectives

This lesson achieves learning objectives:

3. Design front-end user experiences using accepted web design patterns, methods, and information structures.
4. Identify and use strategies of successful visual rhetoric for the web.

#### Alignment

>  The way that the different elements in a design are arranged, usually in relation to a page or document.

##### Grid

        > accuracy in any design.
        Grids allow for vertical and horizontal alignment, calculated for the designer by the web browser;
        it's set and forget!
    </div>
</div>

<Slide>
#### Balance
> Distribution of visual assets by "weight" to create design `symmetry`, `asymmetry`, or `radial symmetry`.
##### Symmetry
<div class = "slide-grid-two-column">
    <div>
        <Image 
            src={symmetry}
            alt="CMPSC 302: Web Design, Course Logo"
            loading="lazy"
        />
    </div>
    <div>
        `Symmetrical` balance distributes page elements at roughly equal distances and hierarchies. When document
        correctly, the distribution of items can create stunning visual effects.
    </div>
</div>
</Slide>

<Slide>
##### Asymmetry
<div class = "slide-grid-two-column">
    <div>
        <Image 
            src={asymmetry}
            alt="CMPSC 302: Web Design, Course Logo"
            loading="lazy"
        />
    </div>
    <div>
        `Asymmetrical` positions elements of different "weights" to create a sense of hierarchical _contrast_, where
        specific elements gain more prominence and attention.
    </div>
</div>
</Slide>

<Slide>
##### Radial Symmetry
<div class = "slide-grid-two-column">
    <div>
        <Image 
            src={radial}
            alt="CMPSC 302: Web Design, Course Logo"
            loading="lazy"
        />
    </div>
    <div>
        `Radial` arrangements call attention to a central element which may be flanked by other elements designed
        to attract visual attention to a specific spot on the screen.

#### Negative Space
> Another name for "white space"; space between elements not populated by design elements. Many designers describe
> negative space as space for "breath."

#### Color Theory
> The use of color to shape understanding and influence perception.
<div class = "slide-grid-two-column">
<div>
        <Image 
            src={colorWheel}
            alt="CMPSC 302: Web Design, Course Logo"
            loading="lazy"

    </div>
